Hi guys,
I am Ishaan Sensharma. This is my personal portfolio website built using React, Vite, and TailwindCSS to showcase projects, technical skills, and professional experience. This portfolio provides an interactive and responsive interface for recruiters and visitors to explore my development work, learn about the developer, and establish contact.
The application focuses on clean UI design, smooth navigation, and responsive layout to ensure accessibility across different devices.

🚀 Project Overview
This project serves as a centralized professional portfolio that highlights development projects, technical expertise, and personal achievements. It is designed to provide recruiters and collaborators with a structured overview of my work.

The website includes multiple sections such as:
1. Introduction and developer profile
2. Technical skills and technologies
3. Featured projects
4. Contact form for communication
The interface emphasizes usability, responsiveness, and modern design principles.

🎯 Objectives
The primary objectives of this portfolio website are:
1. Present development projects in a structured and professional format
2. Demonstrate frontend development skills using modern technologies
3. Provide recruiters with an easy way to evaluate technical capabilities
4. Enable direct communication through a contact form
5. Maintain a responsive layout across desktop and mobile devices

🧰 Technology Stack
The project is implemented using modern frontend technologies:
1. React – Component-based user interface development
2. Vite – Fast build tool and development server
3. TailwindCSS – Utility-first CSS framework for responsive design
4. React Router DOM – Client-side navigation between sections
5. EmailJS – Enables sending messages directly from the contact form
6. React Icons – Icon library for UI components
7. React Toastify – Notification system for form submission feedback
8. React Typing Effect – Typing animation for introduction section
9. React Parallax Tilt – Interactive card animations

✨ Features
Key features implemented in this portfolio include:

👨‍💻 Developer introduction section
📂 Showcase of personal and academic projects
🛠 Skills and technology highlights
📱 Fully responsive UI for multiple screen sizes
📨 Contact form integrated with EmailJS
⚡ Smooth animations and interactive UI elements
🧭 Client-side routing for seamless navigation
